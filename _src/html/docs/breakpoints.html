Breakpoints

  • Breakpoints use the em() to create a responsive build.
  • Add responsive modifiers to trigger different column widths, sorting, or positioning by breakpoint xs:, sm:, md:, lg:, xl:
  • Not all utilities use breakpoint modifiers. See each one in this guide to see if it is available.
