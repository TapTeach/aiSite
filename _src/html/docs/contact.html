Contact

  • Create a contact with a picture, name, title, phone number and email address.
  • Use the Twig {% verbatim %}{% include{% endverbatim %} to show the contact component, components/contact.html.
  • Fill in the parameters.

Parameters

Key Value Default
name The individual's name. --
title The individual's company title. optional
phone number The individual's phone number. optional
email address The individual's email address. optional
img See picture component for parameters --
imgTop A boolean to set the image above the users details. false, optional
imgCir A boolean to change a square image to a circle. false, optional
textCenter A boolean to center align the text. false, optional
