Content Stripes

  • Content that alternates a logo, text on one side and an image or video on the other side.
  • Set the variable to true {% verbatim %}{% set enableLibraryPlyr = true %}{% endverbatim %} on the page when using video, to enable the plyr javascript in the base layout.
Key Value Default
alignRight Aligns the logo and copy to the right and the video or image to the left. false
logo Logo above the copy. See picture component for parameters optional
title Title text under logo. optional
copy Copy text under title. optional
picture Large image. See picture component for parameters optional
video Large video. See video component for parameters optional
