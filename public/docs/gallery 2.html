Gallery

  • Use the Twig {% include to show the gallery component, components/gallery.html.
  • Rely on Contentful content gallery or self hosted images. Contentful gallery only uses id.
  • The self hosted gallery will loop through the number of images provided.
  • Use a thumbnail image and a full size image. Naming convention: thumb-1.jpg, photo-1.jpg. You can choose any word(s) inplace of thumb and photo. The -1 will be automatically added and incremented to the filename with the extension.
  • Set the variable to true {% set enableLibraryLightGallery = true %}, on the page with the gallery, to enable the light-gallery javascript in the base layout.
  • Lightgallery documentation sachinchoolur.github.io.

Where do I find the Contentful asset id?

  • Login to contentful.com.
  • Navigate to "Content" located in the header navigation.
  • Search or look through the list of "Published" galleries. Once desired gallery is selected, click it for additional details.
  • On the gallery detail page, find the "Info" button on the top right corner of the page to retrieve the gallery ID.

Parameters

Key Description Contentful only? Default
srcPath Used for self hosted images with DIR path and beginning of the filename without the extension or -1 number ending. No --
imgExt Self hosted image extension added to the filename. This is optional. No jpg
thumbSrcPath Used for self hosted thumbnail images with DIR path and beginning of the filename without the extension or -1 number ending. No --
thumbExt Self hosted thumbnail image extension added to the filename. This is optional. No jpg
numImgs Sets the total number of imagess. No 1
placeholderSize Define placeholder dimension for lazy loading. The values should be unitless. No 10 10
id Contentful content gallery id. Yes --

Examples

{# Non-Contentful: Gallery images are self hosted #}
{% include "_components/gallery.html" with {
  "gallery": {
    "srcPath": "https://vmlyr-projects.s3.us-east-2.amazonaws.com/style-guide-template-site/images/gallery/photo",
    "imgExt": "jpg",
    "thumbSrcPath": "https://vmlyr-projects.s3.us-east-2.amazonaws.com/style-guide-template-site/images/gallery/thumbs/thumb",
    "thumbSrcPath": "jpg",
    "numImgs": "10",
    "placeholderSize": "200 133"
  }
} %}

{# Contentful: Gallery images via api #}
{% include "_components/gallery.html" with {
  "gallery": {
    "id": "xyYTrr7N8O8SUSmEKmCKW"
  }
} %}
