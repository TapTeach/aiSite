Team

  • Create a team section with pictures, names, titles, and bios.
  • Use the Twig {% include to show the team component, components/team.html.
  • Fill in the parameters and the team modal div.
  • The team modal div will show the bio in a pop over window.
  • You can find photos of individuals on passport.

Parameters

Key Value Default
name The individual's name. --
title The individual's company title. --
bio The individual's biography with html markup. If bio is empty, the hover is disabled and the background color is changed. disabled
picture See picture component for parameters --
