Carousel

  • Carousel slider to showcase your content. There are no limitations to what content can be shown, it can be media or component based.
  • Use carousel component {% embed "_components/carousel.html" %} to pass in slide information.
  • Use these break points for responsive options: 480, 568, 768, 980, 1280, 1450.
Key Type Description
tnsOptions Object {} Any options from the tiny-slider documentation can be used. The tiny-slider option items has the default value of 1 which is recommended for mobile. Use responsive: {} with breakpoints to increase the number of images being displayed in the viewport.
counter Object {} Options relative to the carousel counter. See below.
counter.hide Boolean Default: false
Show/hide the counter.
counter.backdrop Boolean Default: false
Add light gray background to counter.
counter.inverted Boolean Default: false
Inverts text to white for dark backgrounds.
controls Object {} Options relative to the navigation control arrows. See below.
controls.fixed Boolean Default: false
If true, the controls are bottom right aligned opposite of the counter. Otherwise the arrows are vertically center positioned.
iconLeft, iconRight name: icon name. Default is chevron-right-thin
color icon color. Default is white
size icon size. Default is xl
See iconography for working with icons.
