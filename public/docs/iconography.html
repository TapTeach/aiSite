Iconography

  • The iconography in the style-guide utilizes SVG sprites. By utilizing sprites, we are able to target for a specific icon.
  • Icon color is determined by the fill. By default the icons take on the currentColor of the copy.

Available Icons

{#
  - 1st value is the icon name. This field is required.
  - 2nd value is the color. This field is optional and will default to "lg".
  - 3rd value is the size. This field is optional and will default to "currentColor".
#}

{# everything filled out #}
{{ icon('download', 'cyan', 'xs') }}

{# just the icon-name #}
{{ icon('download') }}

{# just the icon-name and size #}
{{ icon('download', '', 'md') }}

Colors

{# black #}
{{ icon('social-linkedin', 'black', 'lg') }}
{# cyan #}
{{ icon('social-linkedin', 'cyan', 'lg') }}
{#
  currentColor
  ------------
  The outside container has a color value of red. Because we are using currentColor,
  the icon is inhering what the parent is set to.
#}
{{ icon('social-linkedin', 'currentColor', 'lg') }}
